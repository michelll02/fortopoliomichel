About Me


Haii Perkenalkan nama saya Michel Michelli, saya lahir diBandung 02 Mei 2002, saya merupakan Mahasiswi STMIK Jabar Semester 5. Dikampus saya menjadi Koordinasi Mahasiswa Jurusan IF (KM), Saya sangat bangga bisa menambah ilmu dan pengetahuan di Learning X. Riwayat prestasi saya sejak dari SMP sampai SMK saya selalu masuk 2 peringkat besar, terakhir kelas 3 SMK saya masuk nominasi Juara umum ke-2. Hingga saat ini saya selalu mendapat ipk diatas rata rata

Saya memiliki keahlian mengoprasikan komputer, menguasai microfoft office, paham tentang UI & UX design, mengetahui beberapa bahasa pemograman dan mengerti tentang sistem basis data. Selain menjadi Seorang Mahasiswi Saya menjadi seorang Freelance dibeberapa perusahaan kecil. Saya memiliki kelebihan semangat belajar yang tinggi, kompeten, rajin, jujur dan dapat bekerja dalam tim maupun individu. Kelemahan saya ada pada kurang percaya diri dan gampang mengeluh.
